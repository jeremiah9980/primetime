# Primetime Player Images

Roster and player profile images should be uploaded here using this naming pattern:

`firstname-lastinitial.jpg`

Examples:

- `khloe-l.jpg` for Khloe Lazo
- `ella-g.jpg` for Ella Gamble
- `kylie-c.jpg` for Kylie Collins
- `zoey-v.jpg` for Zoey Arabella Vela
- `jordynn-w.jpg` for Jordynn Wright
- `araceli-n.jpg` for Araceli Sophia Noveron
- `lyla-t.jpg` for Lyla Terrazas
- `elena-g.jpg` for Elena Alejandra Grimaldo
- `romina-t.jpg` for Romina Alexandra Trevino
- `sophia-h.jpg` for Sophia Texas Hill
- `zeriyah-c.jpg` for Zeriyah Campos

The roster page is `/roster/` and reads these paths through `assets/data/primetime-players.json`.
